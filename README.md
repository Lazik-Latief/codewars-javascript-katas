##  Problem: Counting Sheep

**Description:**  
Given an array of boolean values where:
- `true` represents a sheep present
- `false` represents a missing sheep  

The task is to count how many sheep are present in the array.

---

###  Solution Explanation

- An array contains `true` and `false` values.
- We initialize a counter variable (`count`) to keep track of sheep.
- Using a `for` loop, we iterate through the array.
- Every time we encounter `true`, we increment the counter.
- Finally, we print the total count.

---

###  Key Learnings
- How to iterate over arrays using loops
- How to use conditionals to check values
- Importance of correct loop conditions (`i < array.length`)
- Writing clean and readable JavaScript code
